# CHANGELOG

SoonLink / 溯联的主要变更记录。

标记说明：

- `+特性`：新增能力或可见体验升级
- `=修复`：行为修正、稳定性提升、兼容性收口
- `=文档`：README / CHANGELOG / 说明同步
- `=工程`：构建、发布、仓库边界、CI 等工程治理更新

说明：

- `0.1.x` 到 `0.4.x` 用于承接独立仓库之前的产品演进摘要，帮助后续版本链保持连续。
- 从 `0.5.x` 开始，SoonLink 进入仓库治理阶段，版本、发布和协作流程在仓库内持续留痕。

## [0.0.5] - 2026-05-08

- `+特性` StaticWeb 现在可作为可安装应用壳运行：补齐根作用域 `service-worker`、`manifest.webmanifest`、离线页与统一安装入口
- `+特性` 节点现在可以主动向远端发起传输请求，接收端确认后才会转入既有传输任务链
- `+特性` `transfers` 页新增“来向请求”抽屉，支持轮询查看请求来源、目标路径、备注并手动接受
- `=修复` 主页面、登录页、管理页与控制中心统一补上应用元数据、图标、主题色与 standalone 安全视口设置
- `=修复` 运行时设备记录重新收口 `remembered`、`endpointScheme` 与 `endpointBasePath` 语义；发现到的请求来源只进入候选态，不会因为开放发现而自动长期记忆
- `=工程` 容器 bundle 与 Docker 构建链现在统一从 `dist/` 自包含目录取二进制、配置与 `web` 资源，CI 也会显式校验 `web` 已进入产物
