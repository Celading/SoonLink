# SoonLink Skills Index

This folder is meant for public-facing SoonLink usage patterns, not internal team-only governance.

Good future additions include:

- **CLI snippets** for listing directories, checking version, runtime status, and devices
- **MCP snippets** for letting agents inspect the file tree, runtime state, and tasks
- **issue-report templates** that show how to reduce a bug report to the smallest reproducible shape
- **deployment checklists** for source, Docker, and Homebrew delivery paths
