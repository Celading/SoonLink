# Web Console

SoonLink 的 Web 控制台主要服务于三类操作：

## 1. 文件浏览

- 浏览目录与文件
- 查看文件元信息
- 做文本类内容的轻量预览

## 2. 传输操作

- 上传文件
- 创建目录任务
- 查看任务进度与状态
- 下载已有文件内容
- 使用中转站暂存文件与临时文本

### 中转站文本记录

- 浏览器里粘贴的临时文本会按 UTF-8 `.txt` 记录进入中转缓存，默认适合便签、会议纪要、日志摘录这类短文本。
- 中转站允许远程删除“记录”，删除时只会移除中转记录和临时缓存，不会直接删除设备上的真实文件。
- 选中 `.txt` 记录后，右侧预览区可以直接读取缓存内容。
- 当文本超过预览阈值或读取片段过大时，控制台会提示直接下载，避免浏览器内联渲染过重。
- 文本记录也可以像普通缓存一样回流到当前设备与目录，方便把临时内容转回本地文件树。

## 3. 运行态观察

- 查看设备列表
- 查看当前 session / capability 上下文
- 做基础设置与健康检查

## 适合谁

- 想直接在浏览器里使用 SoonLink 的用户
- 想先用 Web 验证，再切到 CLI / MCP 做自动化的开发者
- 想快速确认某台节点是否工作正常的维护者
