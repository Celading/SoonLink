# Web Console

SoonLink 的 Web 控制台主要服务于三类操作：

## 1. 文件浏览

- 浏览目录与文件
- 查看文件元信息
- 做文本类内容的轻量预览

## 2. 传输操作

- 上传文件
- 创建目录任务
- 查看任务进度与状态
- 下载已有文件内容

## 3. 运行态观察

- 查看设备列表
- 查看当前 session / capability 上下文
- 做基础设置与健康检查

## 适合谁

- 想直接在浏览器里使用 SoonLink 的用户
- 想先用 Web 验证，再切到 CLI / MCP 做自动化的开发者
- 想快速确认某台节点是否工作正常的维护者
